# My Home Page

Welcome to my GitHub home page project! This project is a simple React-based webpage with a navigation bar and some cool features.

## Preview

You can see a live demo of the project [here](https://6549d10ed5c8614da413d82f--tourmaline-cassata-dab3b6.netlify.app/).

## Features

- navigation bar.
- Search box.
- Sign in and Signup buttons.
- Animated SVG images.
